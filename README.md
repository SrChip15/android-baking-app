# Baking App
Android Baking App that allows Udacityâ€™s resident baker-in-chief, Miriam, to share her recipes with the world. The user can select a recipe and see video-guided steps for how to complete it. This app has adaptive UI for phone and tablet devices. Also the user has the ability to *add a recipe as a home screen widget*.

## Tools
* AGP - 3.2
* Android Support Library - 27.1.1
* ViewModel (Android Architecture Component) - 1.1.1
* Butterknife - 8.8.1
* Timber - 4.7.0
* Gson - 2.8.5
* Exoplayer - 2.7.2
* Glide - 4.8.0
